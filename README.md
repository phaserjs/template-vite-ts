# instructions

`pnpm run dev`

`pnpm run build`

## Env settings

`VITE_DEV_MODE`
Is the game running locally in dev mode.

`WIDTH`
The game width. This should match the Figma file.

`HEIGHT`
The game height. This should match the Figma file.

`ENCODE_AUDIO`
Sets if the audio should be encoded into the config.
